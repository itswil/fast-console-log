# 🐸 Fast Console Log

Simple console log shortcut for your JavaScript/TypeScript

## Usage

1. Place cursor on a variable.

2. Use the shortcut:
- Windows/Linux: `Ctrl` `Alt` `l` 
- Mac: `Ctrl` `Option` `l` 

=> `console.log("🐸 count:", count);`

## Requirements

Fast Console Log only works with the following file types:
  - `.js`
  - `.jsx`
  - `.ts`
  - `.tsx`
  - `.mjs`
  - `.cjs`
  - `.mts`
  - `.cts`
  - `.astro`
  - `.svelte`
  - `.vue`
